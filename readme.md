## Getting Started

Follow the steps below to run the application.

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/MikeZone121/MarketHub
   ```
2. Install NPM packages

   ```sh
   npm install or yarn
   ```

3. Run the application
   ```sh
   npm run dev or yarn dev
   ```
